Featured Project: E-commerce Product Price Scraper

This Python-based scraper collects product data from e-commerce websites like Amazon, including product names, prices, ratings, and descriptions. It automates the data collection and stores it in CSV files for easy analysis.

Features

  • Scrapes product details such as name, price, description, and rating.
  • Handles pagination to scrape multiple pages.
  • Saves the scraped data to a CSV file.
  • Includes error handling for timeouts and broken links.

Technologies Used

  • Python
  • BeautifulSoup
  • Requests
  • Pandas
  • Selenium (for dynamic content)
